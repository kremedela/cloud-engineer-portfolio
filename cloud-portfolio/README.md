# Cloud Engineer Portfolio

This repository hosts my **Cloud Engineer Portfolio** using GitHub Pages.  
It demonstrates my skills in:

- Microsoft Azure
- Python
- Terraform
- Web development (HTML, CSS, JavaScript)
- Interactive games and animations

## Website Structure

- `index.html` — Landing page with animated 3D cloud animation
- `cloud-builder.html` — Drag-and-drop cloud builder with Terraform code generation
- `destroy-game.html` — Interactive "Destroy the Link" mini-game
- `python-sandbox.html` — Python in-browser sandbox (PyScript)
- `dashboard.html` — Live Azure dashboard placeholder
- `case-studies.html` — Project case studies and architecture diagrams
- `assets/` — CSS, JS, icons, and Terraform snippets
